Overview

Below are some generated audio samples from the Longformer Encoder-Decoder model. The input to the model is a combination of source-separated Drums, Bass, Vocals and Other stems with the target stem removed. For example, if the target stem is Drums, then the model is provided with the Bass, Vocals and Other stems and predicts the Drum audio stem.
